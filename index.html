    # GreenDesk Documentation Plateforme avancée de gestion, simulation et monitoring d'écosystèmes végétaux.

    Bienvenue sur la documentation officielle de GreenDesk.

    GreenDesk est une application complète pour la gestion, la simulation et le suivi d'espèces végétales, de plantes et de forêts, basée sur Spring Boot et MongoDB. Elle propose une API RESTful, une interface d'administration, des outils de monitoring, et une architecture modulaire adaptée à la recherche, l'enseignement ou la gestion de collections botaniques.

    Fonctionnalités principales¶

    • Gestion CRUD complète des espèces, plantes, forêts
    • Simulation de croissance, saisons, effets environnementaux
    • Monitoring de la santé des plantes (stress, maladies, conditions optimales)
    • API RESTful documentée (Swagger/OpenAPI)
    • Conteneurisation Docker prête à l'emploi
    • Architecture modulaire (Controller, Service, Repository)
    • Support MongoDB local et cloud (Atlas)
    • Tests automatisés (JUnit, SpringBootTest)
